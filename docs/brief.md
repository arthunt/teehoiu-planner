# Romet Rauni tootekirjeldus (proto.txt)

> Puudu ongi see lisaväärtust tekitav osa — süsteem, mis teeks sulle ise
> ettepanekuid, mida remontida, mis järjekorras, ja kui palju see maksab.
>
> Praegu on see kõik Excelis. Keegi vaatab numbreid, teeb oma peas mingi
> otsuse, ja siis vaidleb volikogus. Andmed on olemas — IRI mõõtmised,
> liiklusloendused, kandevuse andmed — aga keegi ei kasuta neid süsteemselt.
>
> Meil oleks vaja midagi lihtsat: valid omavalitsuse, sisestad eelarve,
> ja süsteem ütleb sulle — need teed, selles järjekorras, selle remonditüübiga.
> Punased ülevalt, rohelised alla. Kui eelarve otsa saab, joon alla.

See on see "puuduv osa" mida me demo ajal ehitame.
